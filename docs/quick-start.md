# Mother Brain - Quick Start (Under 2 Minutes)

## What You’ll Get

- A clear project vision
- A research-backed roadmap
- Auto-created skills for repetitive patterns
- Your first task ready to execute

## Prerequisites

- GitHub Copilot CLI installed and configured
- Git installed

## 60-Second Quick Start

```bash
git clone https://github.com/super-state/mother-brain.git
cd mother-brain
gh copilot explain "Use the mother-brain skill"
```

Then select:
- "Yes, start vision discovery"

That’s it. Mother Brain will guide you through vision → roadmap → tasks automatically.

## Next Step

Follow the on-screen prompts to answer 8–12 short questions about your project.  
When it finishes, you’ll have your roadmap and your first task ready.
