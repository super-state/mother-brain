# Mother Brain - Brand Guidelines

## Brand Personality

Mother Brain should feel:
- Cool
- Nerdy
- Retro
- Cutting-edge
- Helpful
- Precise

## Visual Principles

- Retro-tech aesthetics with modern readability
- Dark-first UI with neon accents
- Clear hierarchy and generous spacing
- Minimal clutter, high signal
- Subtle terminal/retro nods (lines, grids, scanlines)

## Color Palette (Retro-Tech)

Primary:
- Brain Magenta: #FF3CAC
- Circuit Purple: #6B1AFF
- Electric Blue: #00AEEF

Neutrals:
- Void Black: #0B0F14
- Deep Space: #151A22
- Signal Gray: #9AA4B2
- Cloud: #F5F7FA

Accent:
- Neon Mint: #A8E6CF
- Warning Amber: #FFB84D

Contrast Guidance:
- Use Cloud or Signal Gray on dark backgrounds
- Avoid magenta on blue for body text
- Reserve neon accents for highlights and calls-to-action

## Typography

Headings (Retro-Tech Display):
- "Orbitron" (Google Fonts)
- "Major Mono Display" (Google Fonts)

Body:
- "IBM Plex Mono" (Google Fonts)
- "Inter" (fallback: system sans-serif)

Usage:
- Headings: Orbitron/Major Mono Display
- Body: IBM Plex Mono or Inter for readability
- Avoid mixing more than 2 fonts on a page

## Banner Usage

- Banner should be lightweight SVG
- Use dark background with neon accents
- Include project name and tagline
- Keep height <= 200px for README

## Voice & Tone

- Crisp, technical, confident
- Avoid marketing fluff
- Use outcome-driven language

## Accessibility

- Target WCAG AA contrast for text
- Do not rely on color alone to convey meaning
- Keep line length under 90 characters for body text
